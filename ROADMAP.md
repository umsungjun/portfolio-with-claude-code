# 개발자 웹 이력서 로드맵

## 기술 스택

- HTML5
- CSS3
- JavaScript (ES6+)
- TailwindCSS

## 프로젝트 구조

```
/
├── index.html          # 메인 이력서 페이지
├── css/
│   └── style.css       # 커스텀 스타일
├── js/
│   └── main.js         # 인터랙션 및 기능
└── assets/
    └── images/         # 프로필 사진, 프로젝트 이미지 등
```

## 이력서 섹션 구성

### 1. 헤더 (Header)
- 이름, 직함
- 연락처 (이메일, 전화번호)
- 소셜 링크 (GitHub, LinkedIn)

### 2. 자기소개 (About)
- 간단한 자기소개 문구
- 프로필 사진

### 3. 기술 스택 (Skills)
- 프론트엔드 / 백엔드 / 기타 기술 분류
- 숙련도 시각화 (프로그레스 바 또는 태그)

### 4. 경력 (Experience)
- 회사명, 직책, 근무 기간
- 주요 업무 및 성과

### 5. 프로젝트 (Projects)
- 프로젝트명, 설명
- 사용 기술, 링크 (GitHub / 데모)

### 6. 학력 (Education)
- 학교명, 전공, 졸업 연도

### 7. 푸터 (Footer)
- 저작권 표시
- 맨 위로 가기 버튼
