# Portfolio with Claude Code

Claude Code를 활용하여 만든 개발자 웹 이력서 프로젝트입니다.

프로젝트 기획부터 구현까지 모든 과정을 Claude Code와 함께 진행했습니다.

## 미리보기

🔗 [https://portfolio-with-claude-code.vercel.app](https://portfolio-with-claude-code.vercel.app)

## 기술 스택

- HTML5, CSS3, JavaScript (ES6+)
- TailwindCSS (CDN)

## 주요 기능

- 싱글 페이지 이력서 (경력, 기술 스택, 학력, 수상/자격증)
- 다크 모드 / 라이트 모드 전환
- 모바일 반응형
- 스크롤 애니메이션 (Intersection Observer)
- SEO 최적화 (Open Graph, Twitter Card)

## 프로젝트 구조

```
├── index.html          # 메인 이력서 페이지
├── css/
│   └── style.css       # 커스텀 스타일
├── js/
│   └── main.js         # 인터랙션
├── favicon.ico
├── CLAUDE.md           # Claude Code 프로젝트 지침
└── ROADMAP.md          # 개발 로드맵
```

## 실행 방법

별도 빌드 도구 없이 브라우저에서 `index.html`을 직접 열어 확인할 수 있습니다.

## Claude Code 활용

이 프로젝트는 [Claude Code](https://claude.ai/code)를 활용하여 다음 작업을 수행했습니다:

- `CLAUDE.md` 프로젝트 지침 작성
- `ROADMAP.md` 개발 로드맵 설계
- PDF 이력서 기반 웹 이력서 구현
- 다크 모드 / 라이트 모드 지원
- SEO 메타 태그 최적화
